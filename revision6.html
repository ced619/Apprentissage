Schéma structure de la page
  1. Header ( En tete )
  2. Nav ( Menu pour inserer les liens de navigation sur le site sous forme de liste a puce)
  3. Section ( morceau de la page dans lequel se trouve ( article et aside )
  4. Article ( Pour mettre un article dans la page web )
  5. aside ( bloc d'info complémentaires au texte
  6. footer ( Pied de la page )
